Privacy Policy

Last updated: August 26, 2025

1. Data We Collect

  • Technical data: User IDs, server IDs, channel IDs, message IDs, command names, timestamps.
  • Configuration data: server settings (e.g., default channels, roles, prefix) set by admins.
  • We do not collect message content unless required for a specific feature, and we store the minimum necessary.

2. Purpose

  • Provide bot functionality, basic stats, and debugging.
  • Improve safety and prevent abuse (spam detection, moderation support).

3. Storage & Retention

  • Configuration data is stored until removed or the bot leaves the server.
  • Logs (if any) are temporary for debugging and deleted periodically.

4. Third Parties

We do not sell data. Limited data may be processed by third-party infrastructure (hosting, logging services) strictly for bot operation.

5. Your Rights

  • Request access or deletion of server-related configuration data.
  • Remove the bot from your server to stop further data collection.

6. Security

We apply reasonable measures to protect data, but no system is fully secure.

7. Underage Users

This bot is intended for users aged 13+ in compliance with Discord policy.

8. Changes

We may update this Privacy Policy. Continued use of the bot means acceptance of the new version.

Contact

If you have questions regarding privacy, contact the bot owner on Discord.
